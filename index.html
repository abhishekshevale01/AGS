Bulk File Rename Application

Description :This project aims to develop a software application that allows users to efficiently rename multiple files at once.
 
Key functionalities:
                                    

  • File Selection: User can choose a folder containing the files they want to rename.
  • Naming Options:
    Basic Renaming: Replace the entire filename with a new name.
    Pattern-Based Renaming: Apply a user-defined pattern to rename files. This pattern can include:
    Original filename elements (e.g., filename stem, extension)
    Sequential numbering
    Prefix or Suffix additions
    Case manipulation (uppercase, lowercase)
  • Preview: The application should display a preview of the new filenames before applying the changes. This allows users to verify the renaming process before committing.
  • File Overwrite Protection: Option to prevent existing files from being overwritten with the new names. This can be achieved through prompts or by adding numbering to avoid conflicts.
  • Batch Processing: Rename multiple files simultaneously based on the chosen options.

Technologies Used: PYTHON, libs- OS, GLOB, SHUTIL
